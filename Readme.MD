<p align="center"><b>Fast, gas-efficient DEX with multi-chain support.</b></p>

**AironDex Multichain** is a decentralized exchange (DEX) designed for low-cost swaps, liquidity pools, and analytics ‚Äî built on BNB Smart Chain (BSC) and compatible with multiple EVM networks.

---

## ‚ú® Features

- ‚ö° **Low-cost swaps**: Optimized for BNB Chain with minimal gas fees  
- üåê **Multi-chain support**: Ethereum, BSC, Sepolia, Arbitrum, Base, and more  
- üíß **Liquidity Pools & LP Rewards**: Uniswap V2-style AMM design  
- üîí **Security controls**: Timelock, pausability, role-based access  
- üìä **Analytics ready**: TVL, volume, TradingView adapter hooks  
- ü™™ **Permit / Permit2 support**: Seamless UX without manual approvals  

---

## üß± Technology Stack

- **Blockchain:** BNB Smart Chain + EVM-compatible chains  
- **Smart Contracts:** Solidity ^0.x.x (OpenZeppelin based)  
- **Frontend:** Next.js + React with wagmi & ethers.js  
- **Dev Tooling:** Hardhat (deploy, verify, tests)
