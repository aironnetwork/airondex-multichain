<p align="center"><b>Fast, gas-efficient DEX with multi-chain support.</b></p>

AironDex Multichain adalah decentralized exchange (DEX) yang berfokus pada low-cost swaps, liquidity pools, dan analytics ‚Äî dibangun di BNB Smart Chain (BSC) dan kompatibel dengan berbagai jaringan EVM lainnya.

## ‚ú® Features

- ‚ö° **Low-cost swaps**: Optimized untuk BNB Chain dengan gas rendah  
- üåê **Multi-chain token support**: Ethereum, BSC, Sepolia, dan jaringan EVM lain  
- üíß **Liquidity Pools & LP Rewards**: Uniswap V2-style AMM  
- üîí **Security Controls**: Timelock, pausability, role-based access  
- üìä **Analytics hooks**: TVL, volume, TradingView adapter  
- ü™™ **Permit / Permit2 support**: UX lebih smooth tanpa manual approve

## üß± Technology Stack

- **Blockchain:** BNB Smart Chain + EVM-compatible chains  
- **Smart Contracts:** Solidity ^0.x.x (OpenZeppelin)  
- **Frontend:** Next.js + React + wagmi / ethers.js  
- **Dev Tooling:** Hardhat (tests, deploy, verify)
